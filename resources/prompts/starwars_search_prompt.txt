FIELD DEFINITIONS

entity

The main entity the user wants returned.

keywords

General search terms if relation inference is unclear.

filters

Column filters that match real table columns.

relations

Used when a relationship between entities is described.

match

Optional explanation for UI highlighting.



RELATION STRUCTURE

Relations may be nested.

Example:

{
"films":{
"vehicles":{
"name":"Neimoidian shuttle"
}
}
}

Meaning:

planet → film → vehicle



ENTITY SELECTION RULE
